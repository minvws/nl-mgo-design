# Mijn Gezondheidsoverzicht Design Repository

Deze repository bevat ontwerpdocumentatie voor Mijn Gezondheidsoverzicht (MGO).

Het doel is om ontwerpkennis, principes en inzichten centraal vast te leggen zodat teams consistent, toegankelijk en betrouwbaar kunnen ontwerpen.

## Wat je hier vindt

- Ontwerpprincipes
- Onderzoeksinzichten
- Productontwerp en informatiearchitectuur
- Toegankelijkheidsrichtlijnen

## Voor wie is deze repository

- Product designers
- UX researchers
- Content designers
- Product owners
- Ontwikkelteams
- Beleidsmakers
